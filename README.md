# Skills

Agent skills for Claude Code and compatible agents.

## Available Skills

### export-session

Export the current Claude Code session to a browsable HTML transcript. Collapses skill content blocks for readability using a [custom fork](https://github.com/stan-voo/claude-code-transcripts) of `claude-code-transcripts`.

```bash
npx skills add stan-voo/skills@export-session
```

### ukrainian-native-editor

Refine Ukrainian prose for nativeness, correctness, and natural rhythm. Removes Russian calques, fixes collocations, enforces 2019 orthography. Includes a 200+ rule reference database.

```bash
npx skills add stan-voo/skills@ukrainian-native-editor
```

## Install All

```bash
npx skills add stan-voo/skills
```
